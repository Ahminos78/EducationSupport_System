# 第一章 绪论

## 什么是数据结构
数据结构是研究非数值计算的程序设计问题中计算机的操作对象以及它们之间关系和操作等的学科。

## 基本概念
- **数据**：所有能输入到计算机中并被计算机程序处理的符号的总称
- **数据元素**：数据的基本单位
- **数据项**：数据的不可分割的最小单位
- **数据对象**：性质相同的数据元素的集合

## 数据结构三要素
1. **逻辑结构**：数据元素之间的逻辑关系
   - 集合结构
   - 线性结构
   - 树形结构
   - 图状结构
2. **存储结构**：数据结构在计算机中的表示
   - 顺序存储
   - 链式存储
   - 索引存储
   - 散列存储
3. **数据的运算**：施加在数据上的操作

## 算法与算法分析
- 算法的五个特性：有穷性、确定性、可行性、输入、输出
- 时间复杂度：T(n) = O(f(n))
- 空间复杂度：S(n) = O(f(n))
